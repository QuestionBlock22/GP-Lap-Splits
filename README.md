## Show Lap Splits after a Grand Prix Race

This is a gecko code that shows the player's lap splits, along with their final rank placement, after a Grand Prix race.

To assemble this code, you may ONLY use PyiiASMH. Download the tool from the releases section (don't clone it) and place the "python3.8.rar" file in the tools folder, then extract it in its place. For more information, see "BUILDING.md."

### Important Pulsar Information
Do not use the gecko code as-is on Pulsar. This is because Pulsar replaces functions in the Time Trial Splits virtual table and replaces the entire page initalization function with its own implementations. To use this gecko code on Pulsar, copy the contents of the "Pulsar" folder to where you saved your Pulsar pack's source code, first and compile. From there you can use the gecko code in its current form.

For use with distributions with closed pack sources or anti-cheat, contact the distribution maintainer to request that this feature be added.
